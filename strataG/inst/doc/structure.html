Running STRUCTURE

Eric Archer

2015-07-29

The strataG package provides an interface to running the program STRUCTURE (Pritchard et al. 2000) as well as some diagnostic measures, the program CLUMPP (Jakobsson and Rosenberg 2007), and plotting. The first step is to make sure that the STRUCTURE and executables are properly installed in the path such that they can be run from any location on a command line. For Windows operating systems, the location of the executables should be specified in the PATH environmental variable. For Macs and UNIX platforms, the executables or links to them can be placed in a folder like /usr/local/bin.

The structureRun function takes a gtypes object and a vector of potential cluster numbers (k.range) to examine. By default, it runs one replicate per value of k, however this can be changed by modifying the num.k.rep argument. Also notable is that the default executable name is structure. If the executable is named something different on your system, this should be changed through the exec argument.

A simple example run is as follows:

# load msat data and create gtypes
library(strataGdevel)
data(dolph.strata)
data(dolph.msats)
msats.merge <- merge(dolph.strata, dolph.msats, all.y = TRUE, description = date())
msats <- df2gtypes(msats.merge, ploidy = 2, id.col = 1, strata.col = 3, loc.col = 5)
rownames(dolph.strata) <- dolph.strata$ids
schemes(msats) <- dolph.strata[, c("fine", "broad")]
msats <- stratify(msats, "fine")

# run 10 replicates of STRUCTURE for k = 2
msat.struct <- structureRun(msats, k.range = 2, num.k.rep = 10)

The result of structureRun is a list for each replicate. Each element in that list is a list with the following elements:

names(msat.struct[[1]])
## [1] "summary"   "q.mat"     "prior.anc" "files"     "label"

The summary element has the following summary values about the replicate run:

msat.struct[[1]]$summary
##           k est.ln.prob    mean.lnL     var.lnL 
##           2       -5864       -5750         226

The q.mat element contains the q-matrix of cluster assignment probabilities:

head(msat.struct[[1]]$q.mat)
##      id pct.miss       orig.pop prob.1 prob.2
## 1 18650        6 Offshore.South      0      1
## 2 18651        0 Offshore.South      0      1
## 3 18652        0 Offshore.South      0      1
## 4 18653        0 Offshore.South      0      1
## 5 18654        0 Offshore.South      0      1
## 6 18655        0 Offshore.South      0      1

Other commonly manipulated parameters for STRUCTURE can be set in structureRun, and are described in detail in the help page and the manual for STRUCTURE.

The Evanno summaries (Evanno et al 2005) can be generated with the evanno function:

msat.struct <- structureRun(msats, k.range = 1:5, num.k.rep = 5)
evanno(msat.struct)

##   k reps mean.ln.k sd.ln.k ln.pk ln.ppk delta.k
## 1 1    5     -6750     1.6    NA     NA      NA
## 2 2    5     -5853     4.5 897.0  900.6  202.37
## 3 3    5     -5856    34.2  -3.6    5.5    0.16
## 4 4    5     -5854    54.4   1.9   28.5    0.52
## 5 5    5     -5881    61.0 -26.6     NA      NA

The results of multiple replicates for a value of k can be combined with clumpp, which returns a q-matrix data.frame:

msat.clmp <- clumpp(msat.struct, k = 3)
head(msat.clmp)
##      id pct.miss       orig.pop prob.1 prob.2 prob.3
## 1 18650        6 Offshore.South  0.666   0.33      0
## 2 18651        0 Offshore.South  0.184   0.82      0
## 3 18652        0 Offshore.South  0.798   0.20      0
## 4 18653        0 Offshore.South  0.120   0.88      0
## 5 18654        0 Offshore.South  0.052   0.95      0
## 6 18655        0 Offshore.South  0.147   0.85      0

These results can then be visualized with structurePlot:

structurePlot(msat.clmp, horiz = T)
