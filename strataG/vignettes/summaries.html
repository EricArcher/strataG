By-Locus Summaries

Eric Archer

2015-09-12

There are several by-locus summary functions available for gtypes objects. Given some sample microsatellite data:

options(digits = 2)
library(strataG)
data(dolph.strata)
data(dolph.msats)
msats.merge <- merge(dolph.strata, dolph.msats, all.y = TRUE, description = date())
msats <- df2gtypes(msats.merge, ploidy = 2, id.col = 1, strata.col = 3, loc.col = 5)
rownames(dolph.strata) <- dolph.strata$ids
schemes(msats) <- dolph.strata[, c("fine", "broad")]
msats <- stratify(msats, "broad")
msats <- msats[, locNames(msats)[1:4], ]

One can calculate the following summaries:

The number of alleles at each locus:

numAlleles(msats)
## D11t D17t   D8 EV37 
##   12   18   13   22

The number of samples with missing data at each locus:

numMissing(msats)
## D11t D17t   D8 EV37 
##    1    2    2    7

which can also be expressed as a proportion of samples with missing data:

numMissing(msats, prop = TRUE)
##   D11t   D17t     D8   EV37 
## 0.0079 0.0159 0.0159 0.0556

The allelic richness, or the average number of alleles per sample:

allelicRichness(msats)
##  D11t  D17t    D8  EV37 
## 0.096 0.145 0.105 0.185

The observed and expected heterozygosity:

obsvdHet(msats)
## D11t D17t   D8 EV37 
## 0.70 0.56 0.47 0.70
exptdHet(msats)
## D11t D17t   D8 EV37 
## 0.75 0.72 0.67 0.83

The percent of alleles that are unique (present in only one sample):

pctUniqueAlleles(msats)
## D11t D17t   D8 EV37 
## 0.25 0.22 0.23 0.14

The value of theta based on heterozygosity:

theta(msats)
## D11t D17t   D8 EV37 
## 0.56 0.54 0.50 0.62

These measures are all calculated in the summarizeLoci function and returned as a matrix. This function also allows you to calculate the measures for each stratum separately, which returns a list for each stratum:
